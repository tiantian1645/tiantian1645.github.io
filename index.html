一些单片机开发技巧

Posted on 2021-11-07

Visual Studio Code 编辑器

  • 支持多光标
  • 括号颜色 或者设置项editor.bracketPairColorization.enabled 另外括号匹配范围设置项也很不错 editor.guides.bracketPairs 更多VsCode 括号效果细节
  • 插件-交换两个光标
  • 插件-输入序列数据
  • 等宽字体-FiraCode
  • 调试STM32 实际上只需完成到第二章编译，即可在VSCode里实现代码补全

附上 c_cpp_properties.json

格式化代码

  • clang-format | VSCode插件 | 配置文件参考 linux

    • 在线工具
    • 本地工具

  • astyle | VSCode插件 | 配置文件参考 mbed-os

  • uncrustify | VSCode插件 | 配置文件参考 FreeRTOS

一开始用的Astyle，后来因为其不支持注释对齐所以换成clang-format，但注释对其后会为了对齐去写无意义的注释。

STM32

  • 串口接收 DMA循环模式 | 空闲判断 ST应用笔记 3109 - 方案2样例
  • 定时器 cookbook 第5章可用于步进电机PWM脉冲生成或者蜂鸣器控制
  • FPU使用注意事项 除了H7其余都只支持单精度浮点数 samples[i]*3.3/4096.0 和 samples[i]*3.3f/4096.0f 会有显著性能区别
  • STM32CUbeIDE中.s文件生成参考 stm32cubeide 使用手册 2.4.4 Temporary assembly file and preprocessed C code
  • 可以使用 __attribute__((optimize("-Ofast")))修饰中断服务函数 来源(阮韶南) https://radiokot.ru/circuit/digital/measure/108/
  • 也可以将优化等级调至O0 __attribute__((optimize("-O0"))) 修饰被调试的函数，避免变量被优化

以上未指明来源的均来源于谷歌搜索
